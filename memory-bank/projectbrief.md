# Project Brief: Instagram Satış Chatbot

## Proje Amacı
Instagram üzerinden kişiye özel çizgi film/animasyon hizmeti satışını otomatikleştiren, müşteriyle profesyonel ve samimi iletişim kuran, sipariş toplama ve ödeme yönetimi yapan bir satış botu geliştirmek.

## Kapsam
- Sık sorulan soruları otomatik yanıtlar
- Kullanıcıdan sipariş için gerekli bilgileri toplar
- Belirlenen iş kurallarına (ödeme sonrası başlama, indirim politikası, revize kuralları vb.) uyar
- Kullanıcı adına hızlı ve etkin iletişim sağlar

## Temel Hedefler
- Manuel mesaj trafiğini azaltmak
- Sipariş sürecini otomatikleştirmek
- Satış ve müşteri deneyimini iyileştirmek
- Basit entegre kuruluma sahip olmak (env ile token/key girişi)

## Çerçeve
- Next.js (backend, Vercel-deploy)
- OpenAI Chat API
- Instagram Graph API (webhook ve DM gönderme)
